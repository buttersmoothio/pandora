# Research

Information gathering and explanations specialist.

## Configuration

```jsonc
"agents": {
  "research": {
    "model": "openai/gpt-5.2",
    "maxOutputTokens": 4096
  }
}
```

All agents share the same config options. See [Configuration Reference](/reference/configuration-reference#aiagents) for details.

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `model` | `string` | *required* | Gateway model ID (e.g. `"openai/gpt-5.2"`) |
| `temperature` | `number` | — | Controls randomness. `0` = almost deterministic, higher values = more random. |
| `maxOutputTokens` | `number` | — | Maximum number of tokens to generate |
| `maxSteps` | `number` | `20` | Maximum tool loop iterations before stopping |

## Purpose

- Answering complex questions
- Explaining concepts
- Comparing alternatives
- Summarizing topics

## Recommended models

| Model | Notes |
|-------|-------|
| `openai/gpt-5.2` | Capable, thorough (recommended) |
| `anthropic/claude-sonnet-4.5` | Great explanations |

## Triggers

The operator delegates to research for prompts like:

- "Explain how..."
- "What is..."
- "Compare X and Y..."
- "Give me an overview of..."
- "What are the pros and cons of..."
