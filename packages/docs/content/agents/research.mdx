# Research

Information gathering and explanations specialist.

## Configuration

```jsonc
"agents": {
  "research": { "model": "openai/gpt-5.2" }
}
```

## Purpose

- Answering complex questions
- Explaining concepts
- Comparing alternatives
- Summarizing topics

## Recommended models

| Model | Notes |
|-------|-------|
| `openai/gpt-5.2` | Capable, thorough (recommended) |
| `anthropic/claude-sonnet-4.5` | Great explanations |

## Triggers

The operator delegates to research for prompts like:

- "Explain how..."
- "What is..."
- "Compare X and Y..."
- "Give me an overview of..."
- "What are the pros and cons of..."
