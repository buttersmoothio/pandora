# Agents

Pandora uses a multi-agent system where a main **operator** can delegate tasks to specialized **sub-agents**.

## How it works

```
User Message → Operator → Response
                  ↓
            (decides to delegate)
                  ↓
              Sub-agent → Back to Operator → Response
```

The **operator** is your main AI assistant. It handles most conversations directly, but when it recognizes a specialized task, it can delegate to a sub-agent:

- Coding question? → Delegates to `coder`
- Need research? → Delegates to `research`
- Need current info? → Delegates to `webSearchNative`

Each sub-agent has focused instructions and can use different AI models.

## Configuration

<Tabs items={['Operator only (minimal)', 'With specialists']}>
<Tabs.Tab>

```jsonc
"agents": {
  "operator": { "model": "google/gemini-3-flash" }
}
```

The operator alone is quite capable for general use. Use a fast, tool-call capable model.

</Tabs.Tab>
<Tabs.Tab>

```jsonc
"agents": {
  "operator": { "model": "google/gemini-3-flash" },
  "coder": { "model": "anthropic/claude-sonnet-4.5" },
  "research": { "model": "openai/gpt-5.2" },
  "webSearchNative": { "model": "perplexity/sonar-pro" }
}
```

You can enable any combination — just include the ones you want.

</Tabs.Tab>

## Choosing models

| Strategy | Example |
|----------|---------|
| Cost-optimized | Fast model for operator, capable models for specialists |
| Task-optimized | Best coding model for coder, search model for research |
| Same everywhere | Use one model for simplicity |

**Recommended setup:**

```jsonc
"agents": {
  "operator": { "model": "google/gemini-3-flash" },
  "coder": { "model": "anthropic/claude-sonnet-4.5" },
  "research": { "model": "openai/gpt-5.2" },
  "webSearchNative": { "model": "perplexity/sonar-pro" }
}
```

The operator handles routing and simple tasks — use a fast, cheap model with good tool-calling. Specialists do the heavy lifting — use capable models there.

## Do I need sub-agents?

**Start without them** if you're just getting started. The operator handles most things well.

**Add sub-agents** when you frequently ask certain types of questions:
- Lots of coding → add `coder`
- Need explanations → add `research`
- Need current info → add `webSearchNative`
