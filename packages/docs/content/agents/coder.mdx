# Coder

Programming specialist for code questions.

## Configuration

```jsonc
"agents": {
  "coder": {
    "model": "anthropic/claude-sonnet-4.5",
    "temperature": 0.2
  }
}
```

All agents share the same config options. See [Configuration Reference](/reference/configuration-reference#aiagents) for details.

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `model` | `string` | *required* | Gateway model ID (e.g. `"anthropic/claude-sonnet-4.5"`) |
| `temperature` | `number` | — | Controls randomness. `0` = almost deterministic, higher values = more random. |
| `maxOutputTokens` | `number` | — | Maximum number of tokens to generate |
| `maxSteps` | `number` | `20` | Maximum tool loop iterations before stopping |

## Purpose

- Writing code
- Debugging
- Code review
- Explaining code

## Recommended models

| Model | Notes |
|-------|-------|
| `anthropic/claude-sonnet-4.5` | Excellent coding (recommended) |
| `anthropic/claude-opus-4` | Best quality, more expensive |

## Triggers

The operator delegates to coder for prompts like:

- "Write a function that..."
- "Debug this code..."
- "Review this PR..."
- "How does this code work?"
- "Convert this to TypeScript..."
