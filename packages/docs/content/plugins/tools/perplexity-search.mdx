# Perplexity Search

AI-powered search using [Perplexity](https://docs.perplexity.ai)'s Sonar model. Instead of returning a list of links, it synthesizes an answer with citations.

<Callout type="info">
  This is separate from using Perplexity as your main model provider on the Config page. This plugin gives the Research agent an additional search tool it can call on demand.
</Callout>

## Setup

1. Get an API key from [perplexity.ai](https://www.perplexity.ai)
2. Set the environment variable and restart Pandora:
   ```bash
   PERPLEXITY_API_KEY=pplx-...
   ```
3. Open the **Plugins** page and toggle **Perplexity Search** on

## Features

- Returns a synthesized answer instead of raw search results
- Includes cited sources with titles and URLs
