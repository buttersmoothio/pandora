# Chat UI Setup

Get the Pandora chat interface running in under 5 minutes.

## Prerequisites

Make sure the core server is running:

```bash
cd packages/core && bun run dev
```

This starts the API on `http://localhost:4111`.

## Start the UI

```bash
cd packages/ui && bun install && bun run dev
```

This starts the Next.js UI on `http://localhost:3000`.

## Open the Chat

Navigate to [http://localhost:3000/chat](http://localhost:3000/chat) in your browser. You'll see an empty conversation with a message input at the bottom.

Try sending a message like "Hello" to verify streaming text works, or "What time is it?" to see tool calls rendered in a collapsible panel.

## What's Included

The chat page renders all AI SDK message part types:

| Part Type | What You'll See |
|---|---|
| Text | Streaming markdown via Streamdown |
| Tool calls | Collapsible panel with parameters and results |
| Reasoning | Collapsible "Thinking..." block (when model supports it) |
| Sources | Collapsible citation links (when model provides them) |

## Configuration

To connect the UI to a different core server, set the `NEXT_PUBLIC_API_URL` environment variable:

```bash
# .env.local
NEXT_PUBLIC_API_URL=https://my-pandora-server.example.com
```
