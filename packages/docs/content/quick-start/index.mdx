# Quick Start

Get Pandora running from zero in under 5 minutes.

<Steps>

### Install & Start

```bash
# Clone and install
bun install

# Start everything (server + UI)
bun run dev
```

This starts the API on `http://localhost:4111` and the UI on `http://localhost:3000`.

Storage uses SQLite — no database setup needed.

### Set a Password

When you open the UI for the first time, Pandora shows a setup screen. Enter a password to secure your instance. All functionalities are unavailable until this is done.

On subsequent visits, you'll see a login screen instead.

### Choose a Model

Open the **Config** page in the UI, pick a provider and model. If the provider needs an API key, you'll see instructions for which environment variable to set.

### Start Chatting

Head to the chat page. You should be setup to start chatting!

</Steps>
