# Quick Start

Get Pandora running from zero in under 5 minutes.

<Steps>

### Install & Start

```bash
# Clone and install
bun install

# Start everything (core server + UI)
bun run dev
```

This starts the API on `http://localhost:4111` and the UI on `http://localhost:3000`.

Storage uses SQLite by default — no database setup needed.

### Set a Password

When you open the UI for the first time, Pandora shows a setup screen. Enter a password (minimum 8 characters) to secure your instance. All API endpoints are locked until this is done.

On subsequent visits, you'll see a login screen instead.

### Choose a Model

Open the **Config** page in the UI, pick a provider and model. If the provider needs an API key, you'll see instructions for which environment variable to set.

### Start Chatting

Head to the chat page. Try sending a message like "Hello" to verify streaming works, or "What time is it?" to see tool calls in action.

</Steps>
