# Configuration API

Manage Pandora's agent configuration programmatically via the REST API.

## Endpoints

### Get Current Config

```
GET /api/config
Authorization: Bearer <token>
```

Returns the fully resolved config object (defaults merged with stored values).

### Update Config

```
PATCH /api/config
Authorization: Bearer <token>
Content-Type: application/json

{
  "identity": { "name": "My Assistant" },
  "models": {
    "operator": {
      "provider": "openai",
      "model": "gpt-4o"
    }
  }
}
```

Accepts a partial config object. Values are deep-merged with the existing config, validated, and persisted to storage.

### Reset to Defaults

```
DELETE /api/config
Authorization: Bearer <token>
```

Deletes the stored config and returns the default values.

## Config Schema

| Section | Description | Fields |
|---|---|---|
| `identity` | Agent name | `name` |
| `personality` | Behavior and tone | `systemPrompt` |
| `models` | LLM configuration | `operator: { provider, model, temperature?, maxTokens? }` |
| `toolPlugins` | Tool plugin management | Keyed by plugin ID: `{ enabled, requireApproval?, ...pluginFields }` |
| `channels` | Channel integrations | Keyed by plugin ID: `{ enabled, ...channelFields }` |
| `agentPlugins` | Agent plugin management | Keyed by plugin ID: `{ enabled, tools?, ...pluginFields }` |
| `agents` | Per-agent configuration | Keyed by agent ID: `{ enabled, model? }` |
| `memory` | Semantic recall | `semanticRecall: { enabled, embedder? }` |

## Available Tools

```
GET /api/tools
Authorization: Bearer <token>
```

Returns all registered tools with their manifests and plugin metadata. Tools are controlled at the plugin level — enable/disable and require approval are set on the plugin, not individual tools.

```json
{
  "tools": [
    {
      "id": "current-time",
      "description": "Get the current date and time in ISO 8601 format",
      "permissions": { "time": true },
      "sandbox": "host",
      "annotations": {
        "readOnlyHint": true,
        "destructiveHint": false,
        "idempotentHint": true
      }
    }
  ],
  "plugins": [
    {
      "id": "tools-websearch",
      "name": "Web Search",
      "envVars": [{ "name": "TAVILY_API_KEY", "required": false }],
      "envConfigured": true,
      "configFields": [],
      "enabled": true,
      "config": {},
      "validationErrors": [],
      "warnings": ["No search backend available..."],
      "toolIds": [],
      "requireApproval": false
    }
  ]
}
```

Each tool object contains:

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Unique tool identifier |
| `description` | `string` | What the tool does |
| `permissions` | `object` | Declared capabilities (`time`, `network`, `env`, `fs`, `random`) |
| `sandbox` | `string` | `'compartment'` or `'host'` |
| `annotations` | `object?` | MCP-compatible UI hints (`title`, `readOnlyHint`, `destructiveHint`, `idempotentHint`) |

## Available Models

```
GET /api/models
Authorization: Bearer <token>
```

Returns all supported providers, their models, and whether the required API key is configured:

```json
{
  "providers": [
    {
      "id": "anthropic",
      "name": "Anthropic",
      "models": ["claude-sonnet-4-20250514", "..."],
      "configured": true,
      "docUrl": "https://docs.anthropic.com",
      "gateway": null,
      "envVars": ["ANTHROPIC_API_KEY"]
    }
  ]
}
```

Use this endpoint to build model selection UIs. Providers with `configured: false` are missing their API key.

## Available Channels

```
GET /api/channels
Authorization: Bearer <token>
```

Returns all registered channel plugins with their config and status:

```json
{
  "channels": [
    {
      "id": "channel-telegram",
      "name": "Telegram",
      "envVars": [{ "name": "TELEGRAM_BOT_TOKEN" }],
      "envConfigured": true,
      "configFields": [
        {
          "key": "ownerId",
          "label": "Owner ID",
          "type": "text",
          "required": true,
          "placeholder": "123456789",
          "description": "Your Telegram user ID..."
        }
      ],
      "enabled": true,
      "config": { "enabled": true, "ownerId": "123456789" },
      "loaded": true,
      "webhook": false,
      "realtime": true
    }
  ]
}
```

Each channel object contains:

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Plugin identifier |
| `name` | `string` | Display name |
| `envVars` | `EnvVarDescriptor[]` | Environment variable descriptors |
| `envConfigured` | `boolean` | Whether all required env vars are set |
| `configFields` | `ConfigFieldDescriptor[]` | UI field descriptors |
| `enabled` | `boolean` | Whether enabled in config |
| `config` | `object` | Current config values |
| `loaded` | `boolean` | Whether the adapter loaded successfully |
| `webhook` | `boolean \| null` | Whether webhook mode is available (`null` if not loaded) |
| `realtime` | `boolean \| null` | Whether realtime mode is available (`null` if not loaded) |

## Available Agents

```
GET /api/agents
Authorization: Bearer <token>
```

Returns all registered agent plugins with their manifests and config state:

```json
{
  "agents": [
    {
      "id": "web-search",
      "name": "Web Search",
      "description": "Search the web for current information...",
      "instructions": "You are a web search specialist...",
      "enabled": true,
      "model": null
    }
  ],
  "plugins": [
    {
      "id": "agent-research",
      "name": "Research",
      "envVars": [],
      "envConfigured": true,
      "configFields": [],
      "enabled": true,
      "config": {},
      "validationErrors": [],
      "agentIds": ["web-search"],
      "tools": []
    }
  ]
}
```

Each agent object contains:

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Unique agent identifier |
| `name` | `string` | Display name |
| `description` | `string` | What the agent does |
| `instructions` | `string` | System instructions |
| `enabled` | `boolean` | Whether enabled in config |
| `model` | `object?` | Model override (`{ provider, model }`) or `null` for operator's model |

Each plugin object contains:

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Plugin identifier |
| `name` | `string` | Display name |
| `envVars` | `EnvVarDescriptor[]` | Environment variable descriptors |
| `envConfigured` | `boolean` | Whether all required env vars are set |
| `configFields` | `ConfigFieldDescriptor[]` | UI field descriptors |
| `enabled` | `boolean` | Whether enabled in config |
| `config` | `object` | Current config values |
| `validationErrors` | `string[]` | Config validation errors |
| `agentIds` | `string[]` | Agent IDs provided by this plugin |
| `tools` | `object[]` | Scoped tool manifests with `{ id, name, description, sandbox, timeout, enabled }` |
