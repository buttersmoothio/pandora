# Pandora vs Others

How Pandora compares to other open-source AI assistants.

## vs OpenClaw

[OpenClaw](https://github.com/openclaw/openclaw) (formerly Clawdbot/Moltbot) is a feature-rich AI assistant.

| Aspect | Pandora | OpenClaw |
|--------|---------|----------|
| **Philosophy** | Simple, hackable | Feature-rich, batteries-included |
| **Codebase** | ~3k lines | Large, comprehensive |
| **AI SDK** | Vercel AI SDK (50+ models) | Custom implementation |
| **Channels** | Telegram, Web | WhatsApp, Telegram, Discord, Slack, Teams, etc. |
| **Agents** | Multi-agent delegation | Single agent with tools |
| **Memory** | Conversation history | Long-term persistent memory |
| **Setup** | Single config file | Dashboard, pairing codes |

### Choose Pandora if:

- You want to understand the entire codebase
- You value simplicity over features
- You want to build exactly what you need
- You prefer Vercel AI SDK compatibility

### Choose OpenClaw if:

- You need many channels out of the box
- You want long-term memory across sessions
- You need computer use / desktop control
- You prefer a larger ecosystem

## Why Pandora exists

Pandora prioritizes:

1. **Simplicity** — Read the whole codebase in an afternoon
2. **Hackability** — Add an extension in minutes, not hours
3. **Modern stack** — Built on battle-tested Vercel AI SDK
4. **Your control** — No cloud dependencies, runs on your machine

If you need maximum features, OpenClaw or similar projects may be better fits. If you want a clean foundation you can fully understand and customize, Pandora is for you.
