# Reference

All types are exported from `@pandorakit/sdk/agents`:

```ts
import type { Agent } from '@pandorakit/sdk/agents'
```

### `Agent`

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | `string` | Yes | Unique agent identifier |
| `name` | `string` | Yes | Human-readable display name |
| `description` | `string` | Yes | What the agent does (used for routing decisions) |
| `instructions` | `string` | Yes | System instructions for the agent |
| `useTools` | `string[]` | No | Tool IDs to pull from the global registry (set from manifest at load time) |
| `modelTools` | `string[]` | No | Model-native tool keys like `'search'` (set from manifest at load time) |

`useTools` and `modelTools` are typically declared in the manifest's `provides.agents` entries rather than in the agent code. They are set automatically at load time.
