# Configuration

Pandora's settings are managed through the **Config** page in the UI. Changes are saved to your database and persist across restarts.

## Models

Choose which AI provider and model Pandora uses. See the [Models](/user-guide/models) page for the full guide.

## Identity

Give Pandora a name. This appears in the UI and is included in the system prompt.

## Personality

The system prompt defines how Pandora behaves — its tone, style, and boundaries. Edit it in the Config page to reshape its character entirely.

## Memory

Let Pandora remember past conversations and bring up relevant context on its own. See the [Memory](/user-guide/memory) page for details.

## Resetting

To reset all settings to their defaults, use the reset option in the Config page. This clears your stored configuration but does not affect your conversation history or password.
