# Configuration

Pandora's settings are managed through the **Config** page in the UI. Changes are saved to your database and persist across restarts.

## Models

Choose which AI model powers your agent. Open the Config page, pick a provider and model, you'll see instructions for which environment variable configurations to set.

You can also adjust:

- **Temperature** — controls randomness (0 = focused, 2 = creative)
- **Max tokens** — limits the length of responses

## Identity

Give your agent a name. This appears in the UI and is included in the agent's self-awareness context.

## Personality

The system prompt defines how your agent behaves — its tone, style, and boundaries. Edit it in the Config page to reshape the agent's character entirely.

## Tools

Tools extend what your agent can do — for example, checking the current time, searching the web, or calling external APIs.

Each tool can be individually **enabled or disabled** in the Tools page in the web UI. Disabled tools are never available to the agent.

Some tools may require **approval before execution**. When enabled, the agent will ask for your confirmation before running the tool, giving you control over potentially sensitive operations.

If you're a developer looking to create your own tools, see the [Custom Tools](/developer-guide/tools) guide.

## Resetting

To reset all settings to their defaults, use the reset option in the Config page. This clears your stored configuration but does not affect your conversation history or password.
