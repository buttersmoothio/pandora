# Exa Search

Web search powered by [Exa](https://exa.ai). Automatically extracts full page text alongside results, making it well-suited for research.

## Setup

1. Get an API key from [exa.ai](https://exa.ai)
2. Set the environment variable and restart Pandora:
   ```bash
   EXA_API_KEY=...
   ```
3. Open the **Plugins** page and toggle **Exa Search** on

## Tool Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `query` | string | yes | The search query |
| `num_results` | integer | no | Number of results, 1--10 (default 5) |

## Features

- Up to 10 results per search (defaults to 5)
- Extracts full page text, not just snippets
- Returns a title, URL, and page content for each result
