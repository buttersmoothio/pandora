# Chat UI

The official Pandora UI is a Next.js app that connects to the core server's streaming API.

## Running

Start both the core server and UI together from the workspace root:

```bash
bun run dev
```

Or separately:

```bash
# Terminal 1 — Core (port 4111)
cd packages/core && bun run dev

# Terminal 2 — UI (port 3000)
cd packages/ui && bun run dev
```

## Connecting to a Remote Server

By default, the UI connects to `http://localhost:4111`. To point it at a different server:

```bash
# .env.local
NEXT_PUBLIC_API_URL=https://your-pandora-server.example.com
```
