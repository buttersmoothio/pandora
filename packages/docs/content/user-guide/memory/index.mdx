# Memory

When enabled, Pandora remembers past conversations and can bring up relevant context on its own — no need to remind it.

## Enabling Semantic Recall

Open the **Config** page and scroll to the **Memory** section:

1. Toggle **Semantic Recall** on
2. Pick a **Provider** and **Model** for embeddings

<Callout type="info">
  The embedding model requires a configured provider API key. For example, selecting an OpenAI model requires `OPENAI_API_KEY` to be set.
</Callout>

That's it. No extra setup is needed — memory works with the default SQLite local database.
