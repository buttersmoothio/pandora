# Tools

Tools extend what your agent can do — checking the current time, searching the web, calling external APIs, and more. Manage them from the **Tools** page in the web UI.

## Enabling a Tool

Each tool belongs to a plugin. To use a tool:

1. Set the plugin's required environment variables (listed on the card)
2. Fill in any required configuration fields and click **Save**
3. Toggle the plugin on

You can also toggle individual tools within a plugin if you don't need all of them.

## Permissions

Each tool displays badges for the capabilities it declares: **Time**, **Network**, **Env**, **Filesystem**, and **Random**.

You can check **Require Approval** on any tool to have the agent ask for confirmation before running it.
