# Web Search Tools

These tools power the `webSearchTool` agent. Configure one, then set it as the `searchBackend`.

## Tavily

AI-optimized web search with high-quality results.

```jsonc
"tools": {
  "tavilySearch": { "apiKey": "tvly-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "tavilySearch"
  }
}
```

| | |
|--|--|
| Get API key | [tavily.com](https://tavily.com/) |
| Free tier | Yes |
| Best for | General search, news, current events |

## Native vs Tool-based Search

Pandora offers two approaches to web search:

| | Native Search | Tool-based Search |
|--|---------------|-------------------|
| Config | `webSearchNative` agent | `webSearchTool` agent + search tool |
| Models | Search-enabled only (Perplexity, Gemini) | Any model |
| Extra cost | None | Search API costs |
| Speed | Faster | Slightly slower |

**Use Native** for simplicity — just set a search model:

```jsonc
"agents": {
  "webSearchNative": { "model": "perplexity/sonar-pro" }
}
```

**Use Tool-based** when you want a specific model (like Claude) to analyze results:

```jsonc
"tools": {
  "tavilySearch": { "apiKey": "..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "tavilySearch"
  }
}
```
