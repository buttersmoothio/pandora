# Web Search Tools

Search backends that power the `webSearchTool` agent. Configure a backend under `ai.tools` with an API key, then reference its config key as `searchBackend` in the `webSearchTool` agent config.

<Callout type="info">
These tools are only needed for **tool-based search** (`webSearchTool`). If you use **native search** (`webSearchNative`) with a search-enabled model like Perplexity Sonar, you do not need any of these backends. See [Web Search Agents](/agents/web-search) for a full comparison.
</Callout>

---

## Tavily

General web search optimized for AI applications.

```jsonc
"tools": {
  "tavilySearch": { "apiKey": "tvly-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "tavilySearch"
  }
}
```

| | |
|--|--|
| Config key | `tavilySearch` |
| Get API key | [tavily.com](https://tavily.com/) |
| Free tier | Yes |
| Best for | General search, news, current events |

## Exa

AI-powered semantic search with advanced filtering and content extraction.

```jsonc
"tools": {
  "exaSearch": { "apiKey": "exa-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "exaSearch"
  }
}
```

| | |
|--|--|
| Config key | `exaSearch` |
| Get API key | [dashboard.exa.ai](https://dashboard.exa.ai/api-keys) |
| Free tier | Yes |
| Best for | Research, finding conceptually similar content |

---

## Perplexity Search

Perplexity Search API with real-time web search. This is the **Search API** (a tool any model can call), which is distinct from Perplexity Sonar models used with `webSearchNative`.

```jsonc
"tools": {
  "perplexitySearch": { "apiKey": "pplx-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "perplexitySearch"
  }
}
```

| | |
|--|--|
| Config key | `perplexitySearch` |
| Get API key | [perplexity.ai/account/api/keys](https://www.perplexity.ai/account/api/keys) |
| Free tier | No |
| Best for | Time-sensitive queries, real-time information |
