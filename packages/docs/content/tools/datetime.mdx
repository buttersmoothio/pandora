# Datetime Tool

Returns the current date and time. No API key needed.

The tool automatically uses the system timezone. You can override it in the config.

## Configuration

```jsonc
"tools": {
  "datetime": {}
}
```

To override the default timezone:

```jsonc
"tools": {
  "datetime": {
    "timezone": "America/New_York"
  }
}
```

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `timezone` | `string` | No | System timezone | IANA timezone (e.g., `America/New_York`, `Asia/Tokyo`). |

## Example uses

- "What time is it?"
- "What's today's date?"
- "What day is it?"

## Response format

The tool returns an ISO 8601 timestamp:

```
2026-02-08T14:30:00Z
```

The AI converts this to a human-friendly format in the response.
