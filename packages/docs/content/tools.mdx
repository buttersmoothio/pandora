# Tools

Tools give your AI agents extra capabilities — like getting the current time, searching the web, or calling APIs.

## How tools work

```
User: "What time is it?"
         ↓
     Operator decides to use datetime tool
         ↓
     Tool returns: "2026-02-08T14:30:00Z"
         ↓
     Operator: "It's 2:30 PM on February 8th, 2026."
```

The AI decides when to use tools — you don't need to ask explicitly.

## Configuration

Enable tools in `config.jsonc`:

```jsonc
"ai": {
  "tools": {
    "datetime": {}
  }
}
```

Tools are then available to all agents automatically.

## Built-in Tools

### datetime

Returns the current date and time. No API key needed.

```jsonc
"tools": {
  "datetime": {}
}
```

**Example uses:** "What time is it?", "What's today's date?", "What day is it?"

## Web Search Tools

These tools power the `webSearchTool` agent. Configure one, then set it as the `searchBackend`.

### Tavily

AI-optimized web search with high-quality results.

```jsonc
"tools": {
  "tavilySearch": { "apiKey": "tvly-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "tavilySearch"
  }
}
```

| | |
|--|--|
| Get API key | [tavily.com](https://tavily.com/) |
| Free tier | Yes |
| Best for | General search, news, current events |

### Exa

Semantic search that finds conceptually similar content.

```jsonc
"tools": {
  "exaSearch": { "apiKey": "exa-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "exaSearch"
  }
}
```

| | |
|--|--|
| Get API key | [dashboard.exa.ai](https://dashboard.exa.ai/api-keys) |
| Free tier | Yes |
| Best for | Research, finding related content |

### Perplexity Search

Real-time search with filtering options.

```jsonc
"tools": {
  "perplexitySearch": { "apiKey": "pplx-..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "perplexitySearch"
  }
}
```

| | |
|--|--|
| Get API key | [perplexity.ai/account/api/keys](https://www.perplexity.ai/account/api/keys) |
| Free tier | No |
| Best for | Time-sensitive queries |

## Native vs Tool-based Web Search

Pandora offers two approaches to web search:

| | Native Search | Tool-based Search |
|--|---------------|-------------------|
| Config | `webSearchNative` agent | `webSearchTool` agent + search tool |
| Models | Search-enabled only (Perplexity, Gemini) | Any model |
| Extra cost | None | Search API costs |
| Speed | Faster | Slightly slower |

**Use Native** for simplicity — just set a search model:

```jsonc
"agents": {
  "webSearchNative": { "model": "perplexity/sonar-pro" }
}
```

**Use Tool-based** when you want a specific model (like Claude) to analyze results:

```jsonc
"tools": {
  "tavilySearch": { "apiKey": "..." }
},
"agents": {
  "webSearchTool": {
    "model": "google/gemini-3-flash",
    "searchBackend": "tavilySearch"
  }
}
```

## Full example

```jsonc
{
  "ai": {
    "gateway": { "apiKey": "your-gateway-key" },
    "tools": {
      "datetime": {},
      "tavilySearch": { "apiKey": "tvly-..." }
    },
    "agents": {
      "operator": { "model": "google/gemini-3-flash" },
      "webSearchTool": {
        "model": "google/gemini-3-flash",
        "searchBackend": "tavilySearch"
      }
    }
  }
}
```
