# Pandora

Your personal AI assistant that you fully control.

Pandora is an open-source AI assistant designed to be simple, hackable, and yours. Connect it to Telegram, use it from the web, or build your own integrations. Add custom tools, create specialized agents, or swap out any component.

## Why Pandora?

**Simple** — A single config file gets you running. No complex setup, no accounts to manage, no cloud dependencies.

**Hackable** — Every part is extensible. Add a tool in 20 lines of code. Create a new chat channel in under 100. The entire codebase is ~3k lines.

**Yours** — Runs on your machine. Your conversations stay private. Use any AI model you want.

## What can it do?

Out of the box, Pandora comes with:

- **Multi-agent system** — A main operator delegates to specialists (coder, research, web search)
- **Any AI model** — Use Claude, GPT-4, Gemini, Llama, or 50+ others via Vercel AI Gateway
- **Telegram & Web** — Chat from your phone or browser with streaming responses
- **Web search** — Built-in support for Perplexity, Tavily, Exa, and more
- **Extensible everything** — Tools, agents, channels, storage — all pluggable
