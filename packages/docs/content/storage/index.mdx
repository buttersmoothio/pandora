# Storage

Storage backends persist conversation history so Pandora remembers previous messages.

## Quick start

<Tabs items={['SQLite (recommended)', 'In-Memory']}>
<Tabs.Tab>

Conversations survive restarts:

```jsonc
"storage": {
  "type": "sqlite",
  "path": "data/pandora.db"
}
```

</Tabs.Tab>
<Tabs.Tab>

For testing (history lost on restart):

```jsonc
"storage": {
  "type": "memory"
}
```

</Tabs.Tab>
</Tabs>

## Full example

```jsonc
{
  "ai": {
    "gateway": { "apiKey": "your-key" },
    "agents": {
      "operator": { "model": "google/gemini-3-flash" }
    }
  },
  "storage": {
    "type": "sqlite",
    "path": "data/pandora.db"
  },
  "channels": {
    "telegram": {
      "token": "bot-token",
      "ownerId": "user-id"
    }
  }
}
```
